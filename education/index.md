## Learning Paths

### 🔵 GEO Practitioner Path
Mathematical and architectural foundations of generative engine optimization.

**Topics:** Entity density optimization · Citation graph topology · RAG pipeline mechanics · Dense passage retrieval alignment · Semantic manifold density · Formal GEO Score

### 🟣 AEO Architect Path
Protocol-level understanding of autonomous agent commerce infrastructure.

**Topics:** Capability discovery protocol · W3C Verifiable Credentials · Tool-augmented generation schemas · Agent-to-agent negotiation · DPoP authentication · Function Calling alignment

### 🟢 AIO Strategist Path
Enterprise-scale infrastructure optimization for AI-mediated marketing operations.

**Topics:** Warehouse-native CDP · Open Policy Agent governance · Graph-native metadata management · FinOps for inference cost · Composable data plane architecture · Compliance as Code
